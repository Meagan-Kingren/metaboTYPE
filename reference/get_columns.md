# Get all column names in a dataframe

Get all column names in a dataframe

## Usage

``` r
get_columns(dataset, exclude = NULL)
```

## Arguments

- dataset:

  combined dataframe

- exclude:

  optional columns to exclude can be entered here as a list
  =c("colum_name", ...)

## Value

column names as a value

## Examples

``` r
if (FALSE) { # \dontrun{
all_cols <- get_columns(df1)
} # }
```
